Graphs - trees

Tree

Definition

Connected and acyclic graph is called a tree

Example
Note
  1. Any vertex of a tree with a degree ≥2 is an articulation
  2. Any edge of a tree is a bridge
  3. Any tree with n vertices has n−1 edges (forest will have less)
  4. Graph is a tree if and only if any 2 vertices can be connected by only 1 circuit

Forest (k-tree)

Definition

If acyclic graph has k connected components, it's called a forest or a k-tree

Example

Minimal spanning tree

Suppose we have the following tree with edge weights

Definition
  1. Weight of the tree: sum of weights of it’s edges
  2. Minimal tree: tree with smallest possible weight

Kruskal’s algorithm

  1. Choose an edge of smallest weight, if it will not make a cycle
  2. Add this edge to the tree, delete it from initial graph
  3. Repeat while possible
  • All edges in ascending order
{a,d}−1{h,f}−1{b,c}−2{g,h}−2{a,b}−3{d,e}−3{f,i}−3{a,e}−4{e,h}−4{h,i}−4{b,e}−5{c,f}−6{d,h}−6{e,c}−8{d,g}−8
  • Construct a new tree (green edges)

Prim's algorithm

  1. Select vertex v1 and add it to set of selected vertices S
  2. Find an edge of minimal weight incident to v1. Say, it is v2
    • Add v2 to S and edge {v1,v2} to tree edge set.
  3. Find an edge of minimal weight incident to any vertex from set S.
    • Add edge to the tree edge set and it’s incident vertex to S if it won't make a cycle
  4. Repeat until all vertices are connected
    1. Select vertex a | S={a}
      • Incident edges: {a,b}−3, {a,e}−4, {a,d}−1
      • @ {a,d} has minimal weight, add it to the tree
    2. S={a,d}
      • Incident edges: {a,b}−3, {a,e}−4, {d,e}−3, {d,h}−6, {d,g}−8
      • @ {a,b} has minimal weight, add it to the tree
    3. S={a,b,d}
      • Incident edges: {a,e}−4, {d,e}−3, {d,h}−6, {d,g}−8, {b,c}−2, {b,e}−5
      • @ {b,c} has minimal weight, add it to the tree
    4. S={a,b,c,d}
      • Incident edges: {a,e}−4, {d,e}−3, {d,h}−6, {d,g}−8, {b,e}−5, {c,e}−8, {c,f}−6
      • @ {d,e} has minimal weight, add it to the tree
    5. S={a,b,c,d,e}
      • Incident edges: {a,e}−4, {d,h}−6, {d,g}−8, {b,e}−5, {c,e}−8, {c,f}−6, {e,h}−4
      • ! {a,e} has minimal weight but will make a cycle, delete it
      • @ {e,h} has minimal weight, add it to the tree
    6. S={a,b,c,d,e,h}
      • Incident edges: {d,h}−6, {d,g}−8, {b,e}−5, {c,e}−8, {c,f}−6, {h,f}−1, {h,g}−2, {h,i}−4
      • @ {h,f} has minimal weight, add it to the tree
    7. S={a,b,c,d,e,f,h}
      • Incident edges: {d,h}−6, {d,g}−8, {b,e}−5, {c,e}−8, {c,f}−6, {h,g}−2, {h,i}−4, {f,i}−3
      • @ {h,g} has minimal weight, add it to the tree
      • @ {h,i} has minimal weight, add it to the tree

Edge elimination from a cycle

  1. Find any cycle
  2. Find an edge in this cycle having biggest weight
  3. Eliminate this edge
  4. Continue until no more cycles left

Storing trees

We can store a tree with n vertices in array of n−2 elements

Tree to array

Suppose we have a tree:

  1. Enumerate all vertices, get a list of rising vertices is: [1, 2, 3, 7, 8, 9] (marked red)

  2. Find a rising vertex with smallest index (in our case it's 1). It is connected to vertex 5, so we add 5 to our array: (Tree[0] = 5) and remove vertex 1

  3. Repeat until 2 vertices are left

    • Remove vertex 2, add 6 to array: [5, 6]

    • Remove vertex 3, add 0 to array: [5, 6, 0]

    • Remove vertex 5, add 6 to array: [5, 6, 0, 6]

    • Remove vertex 6, add 0 to array: [5, 6, 0, 6, 0]

    • Remove vertex 0, add 4 to array: [5, 6, 0, 6, 0, 4]

    • Remove vertex 7, add 4 to array: [5, 6, 0, 6, 0, 4, 4]

    • Remove vertex 8, add 4 to array: [5, 6, 0, 6, 0, 4, 4, 4]

    • Stop conversion

In our case the array will look like this: [5, 6, 0, 6, 0, 4, 4, 4], and the tree will have just 2 connected vertices: 4 and 9

Steiner's point

Definition

Steiner's point will exist if in the triangle there are no angles ≥120o

  1. Find length of edges
    |AB|=(x2−x1)2+(y2−y1)2
  1. Find longest edge (BC)
  2. Construct triangle ΔBCX such that |BC|=|BX|=|CX|
  1. Draw a circle that goes through points B, C, X
    (x−a)2+(y−b)2=R2
  1. Find a point where circle intersects with line AX
    Line equation: x−x1x2−x1=y−y1y2−y1
