Euler path
...

Euler path is path that goes through every edge of a graph exactly once

Note
  • If the degrees of all vertices of the graph are not less than 2, then the graph has at least one Euler cycle
  • A connected undirected graph has an Euler cycle if, and only if the degrees of all vertices of the graph are even numbers
  • If the degrees of exactly two vertices are odd numbers, then the Euler path exists.
    • Its beginning and end are at vertices of odd degrees

Euler graph
...

If a graph has a Euler path and it's a cycle, the graph is called a Euler graph

Note

Edge graph of Euler graph has both Euler and Hamiltonian cycles.

How to find a Euler cycle
...

  1. Take any vertex
  2. Find adjacent vertex . If there are no adjacent vertices - stop
  3. Check if edge is a bridge
    • If edge is not a bridge, add it to the cycle and remove from the graph
    • If edge is a bridge
      • If there are more vertices adjacent to - choose another vertex
      • If there are no more vertices adjacent to - add edge to the cycle and remove it from graph. Go to vertex and repeat
  4. Repeat until no edges left

Hamiltonian path
...

The Hamiltonian path is a simple open/closed circuit that goes through all vertices of a graph exactly once

Theorem
  • If a graph has a bridge, then it has no Hamiltonian cycle.
  • If the graph obtained by removing this bridge has a Hamiltonian cycle, then the initial graph has a Hamiltonian path.
