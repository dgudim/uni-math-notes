Edge graph

Graph's G=(V,E) edge graph is a graph Gb=(Vb,Eb) that has as many vertices as G has edges, and it's vertices are adjacent if corresponding edges of G were adjacent

Info

Edge graph has 12n=1np2(vi)m edges
Where m - amount of graph's G edges
And n - amount of graphs's G vertices
And p(vi) - vertex degree

Stable and dominant (sub)sets

Stable set

Graph's G=(V,E) vertex subset, SV, is called a stable set, if any of it's two vertices are not adjacent

Tldr

Basically, such set has no edges

Example

{c,d} is a stable set

Stability number

Stability number - size of biggest possible stable subset
In out example it's 2

Note

Stability number satisfies inequality
(G)i=1n11+p(vi)Where p(vi) is vertex degree

How to easily find biggest stable subset

  1. Find complement of graph G

G

G

  1. Find biggest complete subgraph ({e,c,d}) in G's complement
  2. This subgraph's vertices will make a stable set in G

Dominant set

Graph's G=(V,E) subset of vertices is called a dominant set, if any vertex NOT from this subset is adjacent to at least one vertex from this subset

Example

{c,b,f} is a dominant set

Dominance number

Dominance number - size of smallest possible dominant set
In out example it's 3

Note

Sometimes taking vertex with biggest degree makes everything worse


Clique

Clique in an undirected graph G=(V,E) is a vertex subset, SV, which edges make a complete graph

Example

{e,d,f} and {a,b,c,d} are cliques
