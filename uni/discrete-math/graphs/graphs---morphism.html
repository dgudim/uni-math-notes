Graphs - morphism

Homomorphism

Definition

Graphs G1=(V1,E1) and G2=(V2,E2) are homomorphic, if there exists such function: f:G1→G2

  1. If v∈V1, then f(v)∈V2
  2. If e∈E1, then f(e)∈E2
  3. If in graph G vertices u and v were adjacent, then in G2 vertices f(u) and f(v) are also adjacent
Tldr

We map one graph onto another, folding if necessary

Theorem
  1. If function f is homomorphism from G1 to G2, then f(G1) is a subgraph of G2
  2. If graph G1 is connected and function f is homomorphism, then f(G1) is also connected
  3. If graph G1 is complete and function f is homomorphism, then f(G1) is also complete

Isomorphism

Definition

Graphs G1=(V1,E1) and G2=(V2,E2) are isomorphic (G1≅G2), if there exists such bijection f:G1→G2, that
∀{u,v}∈E1⟹{f(u),f(v)}∈E2 and
∀{w,y}∈E2⟹{f−1(w),f−1(y)}∈E1
|V1|=|V2|, |E1|=|E2|

Invariants

Definition

Graph functions (vertices, edges, degree sequences, diameter, amount of centers, amount of articulations, etc.), that have same value with all isomorphic graphs are called invariants.

Example

A, B, C are isomorphic to each other, all of them have 3 vertices, 2 edges, same degree sequences, same diameter and amount of centers, same amount of articulations and so on.

Amount of labeled graphs

Let’s |V|=n;, then amount of all possible edges (pairs of 2 different vertices) is:

m=n(n−1)2

So generating function for amount of n-th order labeled graphs is:

Gn(x)=(1+x)m

Amount of all possible labeled graphs (of n-th order): Gn(1)=(1+1)m=2m

Amount of all possible labeled graphs can be calculated approximately:

gn(x)∼2mn!,n→∞
