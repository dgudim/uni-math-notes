Residue classes, divisibility

Residue classes

Definition

If a,b,n - natural numbers, and n divides (a−b) (a/n and b/n have the same remainder)
Then we can write a≡b(mod  n)

Example

10≡1(mod  3), since 3|(10−1)
14≡4(mod  5), since 5|(14−4)

Theorem

Relation ≡ is equivalence relation (reflective, symmetric and transitive)

  1. For all integers a it is true that a≡a(mod  n)
  2. For all integers a and b it is true that: if a≡b(mod  n), then b≡a(mod  n)
  3. For all integers a, b and c it is true that: if a≡b(mod  n) and b≡c(mod  n), then a≡c(mod  n)
